# Simple Camel Application for Testing

## Requires
### Postgresql 10
### Linux(Ubuntu latest)

## Import database table
Source /resource/database

## Build
mvn clean package

## Run on Linux
java -jar 'target/handling-transaction-0.0.1-SNAPSHOT.jar

## Check on Browser
Url : http://localhost:8080/dashboard

## Stop
ctrl+c on running console
